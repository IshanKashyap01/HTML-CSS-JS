# Button Tag

```HTML
<button type="button">
    <img src="image.jpeg" height="10" width="20" alt="image" title="image">
</button>
```

- `<button>` tag can be used to create a generic buttons in the form

- `type` attribute can have the following values:

    1. `button` a generic button with programmable function

    2. `submit` (default) the same function as `submit` with `input`

    3. `reset` the same function as `reset` with `input`

- It also allows you to display an image within it instead of a text
