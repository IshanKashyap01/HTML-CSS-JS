# Grid Alignment

## Row & Column Gap

```CSS
row-gap: <length>;
col-gap: <length>;
gap: <length> | (<row-gap> && <col-gap>);
```

- `row-gap` and `col-gap` specifies the size of the *gutter* b/w rows and columns
respectively

- `gap` is a shorthand for both, where either a different or a common value can
be specified
