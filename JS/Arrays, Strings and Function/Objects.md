# Objects in JavaScript

- JavaScript allows creating objects without creating a corresponding class first

```js
const student = {
    name: 'John Doe',
    class: 'XII',
    section: 'C',
    subjects: ['Physics', 'Chemistry', 'Maths', 'English', 'PE']
    print: function(){...}
}

console.log(student.name, student['class'])
```
